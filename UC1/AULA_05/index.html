Uso dos condicionais IF no JavaScript

O condicional if é utilizado para executar uma ação apenas quando uma condição for verdadeira.


Exemplo:


let idade = 18;

if(idade >= 18){
  console.log("Maior de idade");
}


Nesse exemplo, a mensagem será exibida apenas se a variável idade for maior ou igual a 18.
