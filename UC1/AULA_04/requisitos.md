Sistema:
Bancos

O problema do sistema:
Falhas ao tentar realizar movimentações financeiras


O objetivo do sistema:
Reestabelecer movimentações financeiras de forma segura para o cliente


Regra de negócio:
Criar mecanismos segurança para proteção de contas
bloquear usuários que tentem, sem sucesso, acessar o sistema varias vezes, alertando para entrar em contato com o banco, como medida de segurança.
atualizar e otimizar o sistema sempre que possível


Os usuários do sistema:
Clientes do banco


Requisitos funcionais:
Ser cliente do banco
realizar verificação de identidade (senhas, autentificação por dois fatores, biometria ou reconhecimento facial)




Requisitos não funcionais:
Estabelecer limites diários ou mensais de valores a serem transacionados 
perguntas de segurança para transações acima dos valores usados comumente
